Shopping Website (Front-End Project)

This project is a static front-end shopping website developed using HTML, CSS, and JavaScript. It demonstrates the core concepts of web development by implementing a clean user interface for product browsing and basic client-side interactivity. The website includes a homepage, product listing page, responsive styling, and JavaScript-based UI behavior.
The project focuses on layout design, styling, and user interaction without using backend services or databases, making it suitable for learning and showcasing front-end development skills.
